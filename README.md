# MarksAnalyzer

MarksAnalyzer is a Python-based menu-driven application that helps manage and analyze student marks efficiently using dictionaries. It allows users to input student data, compute statistics, assign grades, and identify performance levels.

This project demonstrates core Python concepts like dictionaries, loops, functions, and conditional logic.

## Features
- Add student marks with validation
- Calculate maximum, minimum, and average marks
- Identify students scoring above 90
- Assign grades based on marks
- Detect failed students
- Display complete student record

## Tech Stack
- Python
- Jupyter Notebook

## How to Run

1. Clone the repository:
   git clone https://github.com/anikagoell/MarksAnalyzer.git

2. Navigate to the project folder:
   cd MarksAnalyzer

3. Run the program:
   python student_marks.py
